# 🗓️ Tiered Events App

A Next.js app with Clerk authentication and Supabase backend that displays events based on user tier.

## 🚀 Features

- Clerk authentication
- Tier-based event filtering
- Supabase integration
- Tailwind UI
- Tier upgrade simulation
